# Образы контейнеров для практической: 
## Установка веб-сервера Apache & PHP / Установка Wordpress и создание БД MySQL.
В каталоге dockerwebservers представлено два Dockerfile, которые собирают образы 2-х контейнров:

1 контейнер разворачивает веб-сервер Apache с поддержкой PHP. В каталоге website1 имеется каталог files, в нем php файл, который необходимо править студенту на свое ФИО.

2 контейнер разворачивает Wordpress, в Dockerfile прописаны раны, которые скачивают необходимые библиотеки, актуальный архив с русскоязычным Wordpressом с официального сайта,  разархивирывает его. 

Также в каталоге files имеется shell-скрипт, который создает пользователя и базу данных, выводит данные переменных в терминал. Данные значения, которые содержатся в скрипте нужно использовать при инсталляции Wordpress (имя бд, пользователь, пароль)
